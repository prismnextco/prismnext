# PRISMNEXT · 프리즘넥스트 주식회사

회사 소개 웹사이트 (단일 HTML 페이지).

## 구성

```
prismnext/
├── index.html   # 전체 페이지 (HTML + CSS + JS 모두 포함)
└── README.md    # 이 문서
```

## 페이지 섹션

- **Hero** — 프리즘/스펙트럼 SVG 애니메이션 + 메인 슬로건
- **About (회사소개)** — 회사 철학, 핵심 가치 3가지
- **Services (서비스)** — 웹 개발 / 앱 개발 / 광고 대행 / 브랜드 마케팅
- **Contact (문의)** — 회사 정보 + 문의 폼 (mailto 방식)
- **Footer** — 사업자 정보

## 보는 방법

`index.html` 파일을 더블클릭해서 브라우저로 열면 됩니다.
별도 빌드나 서버가 필요 없는 정적 페이지입니다.
